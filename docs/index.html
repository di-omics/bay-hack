Evidence contract

Trust is earned at every transition

01 · biology
Expression gate

Replicated TEM-1 signal must clear the organizer threshold over the no-template control.

02 · physical
Verified plate plan

Wells, controls, source positions, replicates, dose factors, and protocol state are checked before dispatch.

03 · measurement
Kinetic evidence

Every assigned well needs a valid trace. Source files are tied to the result by SHA-256.

04 · assay
Z-prime QC

Weak control separation quarantines the plate and blocks the scientific model update.

05 · adaptation
Observed selection

Round 2 records exactly which round 1 scores selected its compounds and factors.

06 · presentation
Safe receipt replay

A valid sealed receipt recreates the evidence story with zero hardware commands.

Challenge-specific and dependency-free

# full two-round modeled fallback plus sealed receipt
$ python -m bayhack.tem1_demo --receipt run_artifacts/tem1-trust.json

# stage dashboard at http://127.0.0.1:8010

$ python -m bayhack.preflight
$ pytest -q
